## 3. 핵심 기능

```python
def get_llm(model_name: str, temperature: float = 0.7) -> Any:
```

```python
def remove_separators(text: str) -> str:
```

## 4. 답변 생성 로직

### 4.1 우선순위
1. 인터넷 검색이 "Perplexity 사용"이면 → Perplexity 검색 사용
2. 인터넷 검색이 "사용 안 함"이고:
   - RAG가 "사용"이고 PDF 파일이 있으면 → RAG 사용
   - RAG가 "사용"이지만 PDF 파일이 없으면 → 경고 메시지
   - RAG가 "사용 안 함"이면 → 직접 LLM 사용

### 4.2 답변 형식 요구사항
모든 답변에 다음 지시 포함:
- 답변은 반드시 헤딩(# ## ###)을 사용하여 구조화
- 주요 주제는 # (H1)로, 세부 내용은 ## (H2)로, 구체적 설명은 ### (H3)로 구분
- 답변은 서술형으로 작성하되 존대말 사용
- 완전한 문장으로 서술
- 구분선(---, ===, ___) 사용 금지
- 취소선(~~텍스트~~) 사용 금지
- 참조 표시나 출처 문구 사용 금지

### 4.3 스트리밍 모드
- RAG 답변: 스트리밍으로 실시간 표시
- 직접 LLM 답변: 스트리밍으로 실시간 표시
- Perplexity: 스트리밍 없음 (완성된 텍스트 표시)
- 스트리밍 중에도 구분선 제거 적용

### 4.4 다음 질문 3개 자동 생성
- 모든 답변 생성 후 자동으로 다음 질문 3개 생성
- 답변 끝에 "### 💡 다음에 물어볼 수 있는 질문들" 섹션 추가
- 번호와 함께 질문 표시
