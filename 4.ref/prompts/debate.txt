# 토론 설계
- 최대 토론 횟수 만큼 토론. 
- 서로 같은 말을 중복하지 않도록 한다. 할말이 없으면 중지.
- 도구 유형은 다음 3개 중 하나 선택.
- 인터넷 검색은 perplexity (sonar-pro) 사용
- perplexity 사용예
	from langchain_perplexity import ChatPerplexity

    	# 공식 문서에서 확인한 유효한 모델명을 명시적으로 지정
   	llm = ChatPerplexity(
        api_key=perplexity_api_key,
        model="sonar-pro"
        llm.invoke("Perplexity API 연결 테스트입니다.")
- 문서검색을 선택하면 각 에이전트 설정에 pdf 문서를 선택할 수 있도록 함
- 도구없음은 LLM 자체적으로 토론 생성.

- 최대 토론 횟수만큼의 찬성, 반대 논리를 제시한 후, 
- 사회자의 정리요약및 개인생각 표시.
- 스트리밍으로 응답을 생성한다.
- 로딩 스피너 추가: 에이전트가 도구를 사용하거나 답변을 생성할 때 스피너를 표시합니다.
