# 역할

너는 공공 물가 데이터를 분석하는 데이터 분석가이자,
Python 기반 데이터 시각화 및 Streamlit 분석 시스템을 구축하는 엔지니어다.

daejeon_price.db SQLite 데이터베이스를 기반으로,
대전시 주요 생필품 물가를 탐색, 비교, 분석, 시각화할 수 있는
"데이터 분석용 Streamlit 프로그램"을 작성한다.

# 프로젝트 구조

프로젝트 루트: ./5.DatabaseSQL/

- data/
  - daejeon_price.db
- code/
  - (여기에 streamlit 분석 프로그램 생성)
- prompts/

# 데이터베이스 정보

파일:
./5.DatabaseSQL/data/daejeon_price.db

테이블:
price_trends

컬럼 구조:
- month (YYYY-MM)
- market_type (SSM, 전통시장, 대형마트, 백화점)
- category (농산물, 축산물, 수산물, 공산품)
- item (품목명)
- spec (규격)
- price_source (조사지점/유형)
- price (number)

# 프로그램 목표

Streamlit 기반  
"대전시 주요 생필품 물가 분석 시스템" 구축

이 프로그램은 단순 시각화가 아니라:

- 데이터 탐색(EDA)
- 월별 동향 분석
- 카테고리/시장 비교
- 변동성 분석
- 이슈 품목 탐색

이 가능한 “분석 도구”여야 한다.

# 필수 기능

## 1. 데이터베이스 연결

- sqlite3 또는 SQLAlchemy 사용
- price_trends 테이블 로드
- 전체 row 수 출력
- 오류 발생 시 사용자 안내 메시지 표시

---

## 2. 사이드바 필터 시스템

다음 조건을 Streamlit sidebar에서 선택 가능하게 구성:

- 월(month) : 멀티 선택
- 시장 유형(market_type) : 멀티 선택
- 카테고리(category) : 멀티 선택
- 품목(item) : 검색 + 선택
- 가격 범위(price) : 슬라이더

필터 변경 시 전체 분석 결과가 실시간 갱신되도록 구성

---

## 3. 데이터 탐색 영역

- 필터링된 DataFrame 테이블 표시
- 현재 선택 조건 요약 출력
- row 수, 품목 수, 카테고리 수 표시
- 컬럼 설명 영역 제공

---

## 4. 핵심 KPI 영역

카드 형태로 시각화:

- 평균 가격
- 최고가 / 최저가
- 선택 품목 수
- 평균 월간 변화율
- 가장 비싼 품목
- 변동성(표준편차) 가장 큰 품목

---

## 5. 시각화 시스템 (핵심)

Streamlit 앱은 단일 그래프가 아니라,
"분석 목적별 시각화 섹션" 구조로 구성한다.

그래프는 Plotly 또는 Altair 사용 (확대, 툴팁, 범례 가능)

### (1) 월별 동향 분석

- 꺾은선 그래프 (Line Chart)
- x = month
- y = 평균 price
- group = category 또는 item 선택 가능
- 여러 품목 동시 비교 가능

목적:
- 가격 추세
- 상승/하락 국면 분석

---

### (2) 시장 유형 비교

- 막대 그래프 (Bar Chart)
- market_type 기준 평균 가격
- category 또는 item 기준 필터 적용

목적:
- 유통 채널별 가격 차이

---

### (3) 카테고리/시장 가격 분포

- 박스플롯 (Box Plot)
- category 또는 market_type 기준

목적:
- 가격 안정성
- 이상치 탐색
- 변동성 비교

---

### (4) 구조 패턴 분석

- 히트맵 (Heatmap)

예시:
- month × category 평균 가격
- month × item 평균 가격

목적:
- 계절성
- 구조적 가격 패턴
- 급등 구간 탐색

---

### (5) 이슈 품목 탐색

- Top-N 막대 그래프

기능:
- 가장 비싼 품목 Top 10
- 가격 변동성 Top 10
- 최근월 기준 급등 품목

---

## 6. 동향 자동 요약 영역

필터 조건과 통계 결과를 기반으로
자동 요약 텍스트 생성

예시:

“2025년 5월부터 11월까지 대전시 농산물 평균 가격은
전반적으로 상승 추세를 보였으며,
특히 배추와 무 품목에서 가격 변동성이 크게 나타났다.
대형마트는 전통시장 대비 평균 가격이 높게 형성되어 있다.”

- 수치 기반 문장 생성
- 평균, 증감률, 분산, Top 품목 활용

---
